#### Gulp Tasks
The default build task
```
gulp
```
Rerun the build task when a file changes
```
gulp watch
```

#### [NPM](https://www.npmjs.com/package/izitoast)

```
npm install izitoast --save
```
#### Yarn
```
yarn add izitoast --save
```
#### Bower
```
bower install izitoast --save
```
